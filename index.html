Response Operatic Characteristic Curve (ROC)

Landing page for roc.js support library. This library can be required client or server side, with or without graphics, i.e. direct calculation of AUC supported.

Computation

All computation takes place in this browser session. No data is transmited to a server, because there is no server supporting this application. The main purpose of this tool is indeed to illustrate how serverless statistical proceedures can migrate to where the data is.
